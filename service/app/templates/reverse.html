Signal Finder

Find an unknown signal in a CAN capture by matching it against a reference you already know. For a button or switch, the quickest way is "Find a control" below.

Find a control (all buses at once)

The simplest way to find a button, switch, or knob. Press Start, then operate the control a few times, tapping Mark (or the spacebar) each time you do. AutoPi listens to every active bus at once and shows the CAN message that reacts to your presses. No reference, no guessing which bus it is on.

Or type a reference by hand

One point per line as time, value (time in the same seconds as the capture). Mark a value na to skip a moment the signal was unavailable. Sweep the value across a range and give at least a dozen points spread over the capture's time range, so the search can tell the real signal apart from ordinary bus noise.
