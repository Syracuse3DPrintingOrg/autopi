Pull a vehicle's whole setup, databases, actions, layout, and simulation, from a central AutoPi server, so a new device does not need each vehicle built by hand. This is a future feature: it has no effect until a server exists and this device is pointed at it.

AI Assist

The Signal Finder works on its own with no account needed. Add a Claude API key here and it can also suggest names and units for the signals it finds and interpret what a message is carrying. Nothing is sent anywhere until you use one of those buttons, and the key stays on this device.
